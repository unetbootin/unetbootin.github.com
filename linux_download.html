UNetbootin - Downloads for Linux

Ubuntu Package

To install UNetbootin from the Ubuntu PPA, run the commands:

sudo add-apt-repository ppa:gezakovacs/ppa
sudo apt-get update
sudo apt-get install unetbootin

Binaries: 32-bit binary 64-bit binary

To run these binaries, download them and run the command chmod +x ./unetbootin-linux, or go to Properties->Permissions and check "Execute"), then start the application by running ./unetbootin-linux

Other Packages (may be outdated): Ubuntu Debian Fedora Suse Arch Gentoo

Sources: Tarball (tar.gz) Github
